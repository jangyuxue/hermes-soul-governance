# user-memory

Agent auto-fills these files based on what you say. No manual editing needed.

| File | Auto-filled when you say... |
|------|---------------------------|
| `preferences.md` | "I like...", "I prefer...", "Your tone should be..." |
| `user-profile.md` | "I am...", "My name is...", "I work as..." |
| `environment-setup.md` | "My system is...", "I use...", "I installed..." |
| `workflows/<name>.md` | "My steps are...", "First I do X, then Y..." |

`.backup/` — auto-created before each write (see SOUL.md Section 5.1).

`workflows/workflow-commands.json` — machine-readable execution steps (see SOUL.md Section 6.3).
